# ⚠ Requisitos

- Yarn / Npm (v7 > x.x)

<br>

# 🔨 Instalacion

Posicionese en la raiz del proyecto y desde la consola digite el siguiente comando

```ssh
yarn install
```

<br>

# 🔑 Scripts

Comandos para levantar cada una de las aplicaciones en local

```ssh
yarn start:website
yarn start:admin
```

Comandos para crear el bundle de cada aplicación

```ssh
yarn build:website
yarn build:admin
```
